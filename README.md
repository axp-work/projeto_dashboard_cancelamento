## Sobre o Projeto
Dashboard interativo criado com **Plotly** para análise de cancelamento de clientes de uma empresa de telecomunicações.

## Visualize o Dashboard
Clique aqui para abrir o dashboard interativo: [dashboard_cancelamento_profissional.html](dashboard_cancelamento_profissional.html)

## Gráficos Incluídos

| Gráfico | Insight |
|---------|---------|
| Cancelamento por Contrato | Contrato mensal tem maior taxa de cancelamento |
| Tempo como Cliente | Clientes que cancelam têm menos tempo de casa |
| Valor da Mensalidade | Clientes que cancelam pagam mensalidades mais altas |
| Serviços Contratados | Serviços de segurança reduzem cancelamento |

pip install pandas plotly
